### To Run the app:
<p>.Just open the console as an administrator, otherwise it may not work.
.Type the command ==> C:\User> npm install  
.after that you will get a file in the root project directory named "node_modulues"
.then we have to install dependencies that we have already in package.json file
.dependencies like:
1.express                   6.passport 
2.ejs                       7.cookie-parser
3.body-parser               8.passport-local 
4.mongoose                  9.method-override
5.connect-flash 
.you can install these dependencies by just type in console ==> C:\User> npm install -g <name-of-the-dependencies>
. then you need to create your own environment variable for the database connections and authorization.
. after all these things you can now run the app in your local host by just type the commands in console => C:\User> node app.js </p>
